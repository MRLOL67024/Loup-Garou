Règles du Jeu - Loup-Garou de Thiercelieux

Dans le jeu du Loup-Garou de Thiercelieux, les joueurs incarnent les habitants d’un village assailli par des loups-garous. Chaque nuit, ces derniers dévorent un villageois et chaque jour, les villageois tentent de découvrir et d’éliminer les loups-garous.

But du Jeu

Pour les Villageois : Éliminer tous les Loups-Garous.

Pour les Loups-Garous : Dévorer tous les Villageois.

Matériel

Des cartes représentant les différents rôles du jeu, un meneur de jeu pour orchestrer les parties.

Personnages

Les Loups-Garous

Chaque nuit, les Loups-Garous se réveillent et désignent une victime à dévorer. Leur objectif est d’éliminer tous les villageois.

Le Villageois

Les villageois n'ont pas de pouvoirs particuliers, mais ils doivent utiliser leur capacité de déduction pour découvrir qui sont les loups-garous et les éliminer.

La Voyante

Chaque nuit, la Voyante peut regarder la carte d'un joueur pour découvrir son rôle. Elle doit utiliser ses informations pour guider les villageois sans se faire démasquer par les loups-garous.

Le Chasseur

Si le Chasseur est éliminé, il peut immédiatement tirer une dernière fois et désigner un joueur qui sera éliminé avec lui.

La Petite Fille

La Petite Fille peut espionner pendant la nuit en ouvrant les yeux pour voir l'identité des Loups-Garous. Cependant, si elle est surprise, elle risque d'être la prochaine victime.

La Sorcière

La Sorcière possède deux potions : une potion de vie pour ressusciter un joueur tué par les Loups-Garous et une potion de mort pour éliminer un joueur. Elle peut utiliser chaque potion une seule fois dans la partie.

Le Cupidon

La première nuit, Cupidon désigne deux joueurs qui tomberont amoureux. Si l'un des amoureux est éliminé, l'autre meurt également de chagrin. Les amoureux doivent défendre leur amour même si cela signifie trahir leur camp.

Le Voleur

Si la carte du Voleur est utilisée, deux cartes supplémentaires sont incluses dans la distribution. Après avoir distribué les cartes, le Voleur peut échanger sa carte contre l'une des deux cartes non distribuées.

Le Maire

Le Maire est élu par les joueurs et, en cas d'égalité lors du vote pour éliminer un joueur, sa voix compte double.

Le Bouc Émissaire

Si le vote pour éliminer un joueur aboutit à une égalité, c'est le Bouc Émissaire qui est éliminé à la place. Le Bouc Émissaire choisit ensuite qui pourra voter le lendemain.

L'Idiot du Village

Si l'Idiot est désigné pour être éliminé lors du vote, il révèle son rôle et reste en jeu, mais il ne peut plus voter. Cependant, il peut encore être tué par les Loups-Garous.

L'Ancien

L'Ancien résiste à la première attaque des Loups-Garous. Si les villageois votent pour l'éliminer, ils perdent leur capacité de vote pour le jour suivant.

Le Corbeau

Chaque nuit, le Corbeau désigne un joueur qui recevra deux votes supplémentaires contre lui lors du vote du lendemain.

Le Salvateur

Chaque nuit, le Salvateur peut protéger un joueur (y compris lui-même) contre l'attaque des Loups-Garous. Un joueur ne peut pas être protégé deux nuits consécutives.

Le Montreur d'Ours

Chaque nuit, le Montreur d'Ours est réveillé avant les Loups-Garous. Si un Loup-Garou est à côté de lui (à sa gauche ou à sa droite), il est averti par le meneur de jeu.

Le Chevalier à l'Épée Rouillée

Si le Chevalier à l'Épée Rouillée est dévoré par les Loups-Garous, il peut éliminer un de ses attaquants avec lui.
